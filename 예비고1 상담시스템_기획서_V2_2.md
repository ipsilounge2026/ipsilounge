# 예비고1 고등학교 지원 전략 & 학습 준비 상담 시스템 기획서 (v2)

---

## 1. 시스템 개요

### 1-1. 목적
- 예비고1 학생의 **고등학교 지원 전략** 수립을 위한 사전 데이터 수집
- 학생의 **현재 학습 준비 상태**를 정량화하여 시각적으로 제시
- 상담사가 데이터 기반으로 **맞춤형 상담**을 진행할 수 있는 환경 구축

### 1-2. 대상
- 중학교 3학년 재학생 (예비고1)
- 학부모 (공동 응답 가능)

### 1-3. 활용 흐름
```
[사전 질문지 작성] → [데이터 자동 분석] → [시각화 리포트 생성] → [상담사 리뷰] → [대면/비대면 상담]
```

### 1-4. 카테고리 구성 및 설문 흐름

**설계 원칙:**
1. 쉬운 것 → 목표 → 무거운 데이터 → 환기 → 무거운 데이터 (점진적 진입)
2. 목표(진로)를 먼저 파악해야 이후 성적·선행 데이터에 맥락이 생김
3. 성격이 다른 무거운 입력(성적 데이터 vs 자가진단) 사이에 가벼운 질문 삽입 (피로 분산)

| 카테고리 | 내용 | 피로도 | 소요 시간 |
|---------|------|--------|----------|
| A | 기본 정보 (이름, 학교, 희망 고교) | 💚 쉬움 | 2~3분 |
| B | 진로 & 대입 방향성 (진로, 전형, 우선순위) | 💚 가벼움 | 3~5분 |
| C | 중학교 성적 (학기별 원점수, 강점/약점) | 🟡 무거움 | 8~12분 |
| D | 학습 습관 (스케줄, 계획, 오답, 문제 해결) | 💚 보통 | 5~7분 |
| E | 과목별 고등 준비율 (수학/영어/국어/과학 자가진단) | 🟡 무거움 | 10~15분 |
| F | 비교과 & 역량 (동아리, 수상, 리더십, 강점) | 💚 가벼움 | 3~5분 |
| G | 학부모 관점 (선택적 별도 섹션) | 💚 가벼움 | 3~5분 |

**피로도 흐름:**
```
💚💚💚 💚💚💚 🟡🟡🟡🟡🟡🟡 💚💚💚💚💚 🟡🟡🟡🟡🟡🟡🟡🟡 💚💚💚 💚💚
A       B       C(성적)        D(습관)      E(자가진단)         F      G
                 ↑ 무거움       ↑ 환기       ↑ 무거움            ↑ 가벼움
```

## 3. 시각화 데이터 설계

### 3-1. 종합 준비도 — 5각형 레이더 (영역별 독립 100점)

5개 영역 각각 100점 만점으로 독립 산출, 가중 평균 총점은 없음

| 영역 | 산출 근거 | 등급 |
|------|-----------|------|
| 학업 기초력 | C1~C9 | S/A/B/C/D |
| 학습 습관 & 자기주도력 | D1~D5 | S/A/B/C/D |
| 교과 선행도 | E1~E4 | S/A/B/C/D |
| 진로 방향성 | B1~B4 | S/A/B/C/D |
| 비교과 역량 | F1~F6 | S/A/B/C/D |

등급: S(90~100) / A(75~89) / B(55~74) / C(35~54) / D(0~34)

### 3-2. 과목별 성적 추이 (원점수 기준)
- 전과목 전체 추이 라인 차트
- 과목별 개별 추이 라인 차트
- 자유학기제 구간 점선 표시

### 3-3. 과목별 고등 준비율
E1~E4 데이터를 기반으로 수학/영어/국어/과학 각각의 고등학교 학습 준비 상태를 100% 기준으로 시각화.
과목별 총 준비율 바 차트 + 하위 항목(기초/선행/역량 등) 분해 표시.
과목을 탭하면 하위 항목별 상세 비율 확인 가능.

### 3-4. 학습 습관 매트릭스
현재 습관 분석 (자기주도 비율, 계획 실행력, 오답 관리, 문제 해결 적극성)

### 3-5. 고등학교 유형별 적합도
희망 유형과 현재 역량 매칭 바 차트

### 3-6. 대입 전형 준비 로드맵 (학생 맞춤형)

**생성 프로세스:**
```
[학생 설문 데이터] → [자동 분석·초안 생성] → [상담사 검토·수정] → [최종 리포트 확정] → [학생/학부모 전달] → [추후 상담 시 진행 체크]
```

**4단계 × 6트랙 구조:**

| 시기 | 핵심 테마 |
|------|-----------|
| Phase 0: 지금~입학 전 | 기초 보완 & 습관 형성 |
| Phase 1: 고1-1학기 | 고등학교 적응 & 첫 내신 |
| Phase 2: 고1-2학기 | 진로 구체화 & 심화 탐색 |
| Phase 3: 고2 이후 | 전형 전략 본격화 (방향 제시) |

| 트랙 | 내용 |
|------|------|
| 📘 교과 학습 | 과목별 학습 목표·선행 계획 |
| 📋 내신 전략 | 지필·수행평가 대비 |
| 📝 세특 | 교과 연계 탐구 주제·활동 |
| 🎯 수능/모의고사 | 목표 등급·취약 유형 보완 |
| 🏫 비교과 | 동아리·자율·진로활동 |
| 🔧 습관 개선 | 자기주도력·오답관리 등 |

### 3-7. 상담사 검토/수정 범위

**기본 원칙:**
- 학생 설문 데이터 기반 자동 분석·초안 생성 → 상담사가 **모든 초안 내용을 검토·편집** 가능 → 최종 리포트 확정
- 상담사가 수정한 내용이 최종 리포트에 반영됨

**상담사 편집 가능 항목:**

| 구분 | 편집 대상 | 편집 방식 |
|------|-----------|-----------|
| 점수/등급 | 5개 영역별 자동 산출 점수·등급 | 점수 직접 수정 → 등급 자동 재산출 |
| 레이더 차트 | 점수 수정 시 자동 반영 | 점수 수정으로 간접 편집 |
| 성적 추이 분석 코멘트 | 자동 생성된 추이 해석 | 텍스트 직접 편집 |
| 과목별 고등 준비율 분석 | 자동 산출 준비율 + 코멘트 | 준비율 수치 수정 + 코멘트 편집 |
| 학습 습관 분석 코멘트 | 자동 생성된 습관 진단 | 텍스트 직접 편집 |
| 고등학교 적합도 | 자동 매칭 결과 + 코멘트 | 적합도 조정 + 코멘트 편집 |
| 맞춤 로드맵 (3-6) | 4단계×6트랙 자동 초안 | 각 셀 내용 자유 편집 (추가/삭제/수정) |

**상담사 코멘트 (리포트 미반영):**

| 항목 | 용도 | 표시 위치 |
|------|------|-----------|
| 상담사 메모 | 상담 시 확인할 포인트, 주의사항, 질문 사항 등 | 상담사 페이지에서만 표시 |
| 상담 전 체크리스트 | 이 학생 상담 전 미리 검토할 사항 정리 | 상담사 페이지에서만 표시 |

- 상담사 코멘트는 상담 준비·진행용이며, 학생/학부모에게 전달되는 리포트에는 **포함되지 않음**
- 상담사 페이지에서 리포트 내용과 함께 상담사 코멘트가 나란히 표시되어, 상담 시 참고 가능

## 6. 고1 전환 시 데이터 연계

### 6-1. 전환 개요

예비고1 학생이 고등학교에 입학하면, 예비고1 상담 데이터를 고등학교 T1(사전 설문)에 **Delta 방식**으로 연계한다.
- 기존 데이터를 새 설문에 매핑하여 사전 입력 (pre-fill)
- 학생은 기존 내용을 확인하고 **변경/추가 사항만 수정**
- 고등학교 T1 전용 영역(모의고사, 내신 세부 등)은 **새로 입력**

### 6-2. 카테고리 매핑

| 예비고1 카테고리 | 고등학교 T1 카테고리 | 전환 방식 |
|-----------------|---------------------|-----------|
| A (기본 정보) | A (기본 정보) | 자동 매핑 (이름, 학교 등 기본 정보 이관) |
| B (진로 & 대입 방향성) | E (진로·전형 전략) | 자동 매핑 + 확인·수정 (진로, 전형 이해도 등) |
| C (중학교 성적) | — | 참고 데이터로 보존 (고등학교 내신과 별도 관리) |
| D (학습 습관) | D (학습 습관·전략) | 자동 매핑 + 확인·수정 (D1~D5 거의 동일 구조) |
| E (과목별 고등 준비율) | **보존 (비교 데이터)** | 아래 6-3 참조 |
| F (비교과 & 역량) | — | 참고 데이터로 보존 (고등학교에서는 실제 학생부 기록으로 대체) |
| G (학부모 관점) | F (학부모 설문) | 자동 매핑 + 확인·수정 |
| — (신규) | B (내신 성적) | **추가 입력** (고등학교 실제 성적) |
| — (신규) | C (모의고사) | **추가 입력** (고등학교 모의고사 응시 후) |

### 6-3. 예비고1 E영역 보존 (과목별 역량 진단 비교)

예비고1 E영역(과목별 고등 준비율)은 고1 전환 후에도 **삭제하지 않고 보존**한다.

**보존 목적:**
- 중학교 때 자가 진단한 과목별 자신감·흥미와 고등학교 실제 성적을 비교하여 상담 자료로 활용
- 예: "중학교 때 수학 자신 있다고 했는데, 고1 내신은 어떤가요?" → 변화 추적 가능
- 예: "중학교 때 과학 어려워했는데, 고등학교에서 선택과목을 어떻게 정했나요?" → 진로 변화 상담

**표시 방식:**
- 고등학교 상담 리포트에서 "중학교 자가진단 vs 고등학교 실제" 비교 섹션으로 활용
- 상담사 페이지에서 참고 자료로 표시 (리포트에는 상담사 판단에 따라 선택적 포함)

### 6-4. 전환 프로세스

```
[예비고1 상담 완료]
       ↓
[고등학교 입학 확인] ← 관리자가 학생 상태를 "고1"로 전환
       ↓
[고등학교 T1 설문 자동 생성]
  ├── 예비고1 데이터 매핑 (6-2 기준) → 사전 입력됨
  ├── 예비고1 E영역 → 비교 데이터로 보존
  └── 고등학교 전용 영역 (B 내신, C 모의고사) → 빈 상태로 추가
       ↓
[학생이 T1 설문 확인·수정·추가 입력]
       ↓
[고등학교 상담 시스템으로 진행]
```

### 6-5. 전환 시 유의사항

| 항목 | 내용 |
|------|------|
| 전환 시점 | 관리자가 수동으로 전환 (자동 전환 X — 입학 취소, 유예 등 예외 대응) |
| 원본 보존 | 예비고1 설문 원본은 별도 보존 (고1 데이터와 독립) |
| 수정 이력 | Delta 수정 시 변경 전/후 값 모두 기록 |
| 미전환 학생 | 예비고1 상담만 받고 고등학교 상담을 이용하지 않는 경우, 데이터는 예비고1 상태로 유지 |

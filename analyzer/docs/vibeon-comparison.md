## 1. 바이브온 리포트 구조 (40페이지, 3파트)

### Part 1. 학교생활기록부 진단 (p.6~18)

2. 목표 대학 진단 (서류합격예측 적정권/서류합격률 60-70%)
3. 학교생활기록부 총점 (300점 만점, 예: 242.85/300)
4. 학교생활기록부 역량별 점수 (학업역량/진로역량/공동체역량 막대그래프)
5. 학업역량 점수 (점수 + 지원적정구간 비교 + 좋은 평가 문장 발췌)
6. 진로역량 점수 (동일 구조)
7. 공동체역량 점수 (동일 구조)
8. 유형별 내신 성적 현황 (전교과/국영수사과/국영수과/국영수 분류별 학기별 표 + 추이 그래프)
9. 전체 내신 성적 (교과별 내신 평균등급 순위 + 이수과목 목록)
10. 전공 연계 핵심과목 및 권장과목
11. 전공 연계 과목 이수 현황 (핵심과목 이수 여부 + 전공 vs 전체 평균 비교)

12. 학적

18. 진로활동 (동일 구조)
19. 세부능력 및 특기사항 (적극성/탐구정신/전공진로역량 점수 + 과목별 좋은 평가 문장 + 역량별 분류)

### Part 3. 맞춤형 보완점 (p.29~39)

26. 학업역량 진단 및 보완법 (잘한 항목/보완 항목/필수 보완 항목 + 과목별 상세 보완 전략)

## 2. 바이브온 채점 체계

### 총점
- 300점 만점 (학업역량 + 진로역량 + 공동체역량 각 100점)
- 지원적정구간 (전교과 1.72~2.12등급) 기준 상위 % 표시
- 전체 상위 % 표시

### 역량별 세부 평가
- **학업역량**: 학업성취도, 학업태도, 탐구력
- **진로역량**: 전공 관련 교과 이수 노력, 전공 관련 교과 성취도, 진로 탐색 활동 및 경험
- **공동체역량**: 협업과 소통능력, 나눔과 배려, 성실성과 규칙준수, 리더십

### 창체 평가 기준
- 기록의 충실성 / 학업역량 / 공동체역량 3가지로 우수/양호/보통 평가
- 분량 비율: 최대 기재 가능 분량 대비 전체 분량 비율 + 개인기록 비율

### 등급 분류
- 우수 / 양호 / 보완 필요 3단계

### 비교군 데이터
- 전국 2,400개 고교를 학업성취도와 입시결과로 11그룹 분류
- 해당 그룹 내 상대적 위치 표시

## 7. 입시라운지에 추가하면 좋을 기능 (우선순위)

### 즉시 추가 가능 (구현 난이도 낮음)
1. **창체 분량 비율 분석** - 최대 가능 분량 대비 실제 기록 비율 계산
2. **학생부 유형 분류** - 분석 결과 기반 유형 라벨링 (예: "심화탐구형", "다재다능형")
3. **좋은 평가 문장 발췌** - 세특/행특에서 특히 좋은 평가를 받은 문장 하이라이트
4. **추천 도서** - 전공 관련 도서 리스트 (Claude가 생성 가능)

### 중기 추가 (구현 가치 높음)
5. **탐구활동 추천 주제** - 학생부 분석 기반 + 전공 연계 구체적 탐구 주제 제안
6. **AI 전공 추천** - 학과 미지정 학생에게 적합 전공 3개 추천 + 근거
7. **역량별 보완법 상세화** - 현재 보완점을 과목별 구체적 액션 플랜으로 확장
8. **워드클라우드** - 학생부 키워드 시각화 (Python wordcloud 라이브러리)

### 장기 (데이터 축적 필요)
9. **비교군 데이터** - 합격 데이터 축적 후 비교군 생성
10. **합격예측 정교화** - 축적 데이터 기반 서류합격률 예측

## 8. 입시라운지의 차별화 전략

바이브온 대비 입시라운지가 더 강한 영역:

1. **세특 분석 깊이**: 바이브온은 "좋은 문장 발췌" 수준이지만, 입시라운지는 7개 루브릭으로 과목별 상세 채점 + 강점/보완점 서술
2. **연계성 분석**: 바이브온에 없는 "과목 간 연계성", "창체-세특 연계성" 분석으로 차별화
3. **성장 스토리**: 학년 간 성장 서사를 유형화하여 분석 (일관심화/관심확장/전환성장/단절)

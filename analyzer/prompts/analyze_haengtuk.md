# 행특 분석 프롬프트

아래 학생의 **행동특성 및 종합의견(행특)** 텍스트를 학년별로 분석하여 루브릭에 따라 채점하세요.

## 평가 항목 (각 20%, 1~10점)

| 항목 | 9~10점 | 5~6점 | 1~2점 |
|------|--------|-------|-------|
| ① 활동의 구체성 | 특정 상황·행동·결과가 에피소드로 서술 | 활동명은 있으나 구체적 장면 부족 | "~한 학생임" 식 나열 |
| ② 동기-과정-결과 | 세 요소 모두 구체적 연결 | 일부만 있음 | 단순 사실 나열 |
| ③ 성장 변화 | 1→2→3학년 뚜렷한 역량 심화·확장 | 변화는 있으나 방향성 불명확 | 매년 비슷하거나 일관성 없음 |
| ④ 인성·공동체 역량 | 구체적 상황에서 에피소드로 서술 | 키워드는 있으나 근거 부족 | 인성 관련 언급 거의 없음 |
| ⑤ 분량·밀도 | 충분한 분량(500자 근접) + 다양한 활동 + 구체적 | 적정 분량이나 내용 표면적 | 1~2줄 수준 형식적 기술 |

※ 7~8점: "9~10점" 기준에 부분적으로 도달. 3~4점: "5~6점" 보다 약하나 "1~2점" 보다는 구체성 있음.

## 출력 형식 (JSON)

```json
{
  "학년별": {
    "1": {
      "scores": {
        "활동의구체성": {"score": 8, "근거": "학급회장으로서의 구체적 에피소드..."},
        "동기과정결과": {"score": 6, "근거": "과정은 서술되나 동기 부족..."},
        "성장변화": {"score": 6, "근거": "1학년이므로 성장 추이 판단 제한..."},
        "인성공동체역량": {"score": 8, "근거": "배려 사례가 구체적..."},
        "분량밀도": {"score": 8, "근거": "450자 내외로 충실..."}
      },
      "강점": "학급 리더로서의 구체적 에피소드가 잘 드러남",
      "보완점": "학업 관련 성장이나 탐구 역량에 대한 언급이 부족"
    },
    "2": { ... },
    "3": { ... }
  },
  "growth_trend": "1학년 리더십 중심 → 2학년 학업 심화 → 3학년 진로 구체화로 뚜렷한 성장 흐름"
}
```

## 주의사항
- 학년별 독립 채점 후, 학년 간 성장 추이를 별도 서술
- ③ 성장 변화는 전학년 맥락에서 종합 판단 (1학년은 기준점으로 활용)
- 강점/보완점은 학부모 상담에 바로 활용할 수 있는 구체적 코멘트로 작성

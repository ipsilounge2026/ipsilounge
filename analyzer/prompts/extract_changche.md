# 창의적 체험활동 (창체) 추출 프롬프트

아래 학생부 이미지에서 **창의적 체험활동상황** 데이터를 추출하세요.

## 추출 대상

### 1. 자율활동 / 동아리활동 / 진로활동
학년별 특기사항 텍스트를 **원문 그대로** 추출합니다.

### 2. 봉사활동
학년별 봉사활동 시수를 추출합니다.

## 출력 형식 (JSON)
```json
{
  "changche": {
    "자율": {
      "1": "1학년 자율활동 특기사항 원문...",
      "2": "2학년 자율활동 특기사항 원문...",
      "3": "3학년 자율활동 특기사항 원문..."
    },
    "동아리": {
      "1": "1학년 동아리활동 특기사항 원문...",
      "2": "2학년 동아리활동 특기사항 원문...",
      "3": "3학년 동아리활동 특기사항 원문..."
    },
    "진로": {
      "1": "1학년 진로활동 특기사항 원문...",
      "2": "2학년 진로활동 특기사항 원문...",
      "3": "3학년 진로활동 특기사항 원문..."
    }
  },
  "volunteer_hours": {
    "1": 20,
    "2": 15,
    "3": 10
  }
}
```

## 주의사항
- 텍스트는 원문 그대로 추출 (요약하지 말 것)
- 학년 키는 "1", "2", "3" 문자열
- 봉사활동은 시수(숫자)만 추출
- 자율동아리는 대입 미반영이므로 정규 동아리만 추출
- 해당 학년에 데이터가 없으면 빈 문자열 "" 또는 0

# 세부능력 및 특기사항 (세특) 추출 프롬프트

아래 학생부 이미지에서 **교과학습발달상황의 세부능력 및 특기사항** 텍스트를 추출하세요.

## 추출 대상
각 학기별, 과목별 세특 기술 내용을 **원문 그대로** 추출합니다.

## 출력 형식 (JSON)
```json
{
  "1-1": [
    {
      "과목명": "국어",
      "텍스트": "수업 시간에 적극적으로 참여하며..."
    },
    {
      "과목명": "수학",
      "텍스트": "함수의 극한 단원에서..."
    }
  ],
  "1-2": [...],
  "2-1": [...],
  "2-2": [...],
  "3-1": [...],
  "3-2": [...]
}
```

## 주의사항
- 텍스트는 원문 그대로 추출 (요약하지 말 것)
- 과목명은 성적 데이터의 과목명과 일치하도록 표기
- 학년-학기 키는 "1-1" ~ "3-2" 형식
- 해당 학기에 세특이 없으면 빈 배열 []
- 띄어쓰기, 마침표 등 원문 서식 유지

# 교과학습발달상황 (성적) 추출 프롬프트

아래 학생부 이미지에서 **교과학습발달상황(성적)** 데이터를 추출하세요.

## 추출 대상
각 학기별로 모든 과목의 다음 정보를 추출합니다:

| 필드 | 설명 | 예시 |
|------|------|------|
| 과목명 | 과목 이름 | "국어", "수학Ⅰ", "미적분Ⅱ" |
| 교과군 | 교과 영역 | "국어", "수학", "영어", "사회", "과학", "체육", "예술", "기술가정", "제2외국어", "한문", "교양" |
| 과목유형 | 이수 구분 | "공통", "일반선택", "진로선택", "융합선택" |
| 학점 | 이수 학점 | 3 |
| 원점수 | 원점수 | 85 |
| 과목평균 | 과목 평균 점수 | 72.3 |
| 성취도 | A/B/C/D/E (5등급제) | "A" |
| 성취도별분포비율 | A/B/C/D/E 비율 | "15.2/25.3/30.1/20.0/9.4" |
| 석차등급 | 1~5등급 (진로선택은 null) | 1 |
| 수강자수 | 수강 인원 | 150 |

## 출력 형식 (JSON)
```json
{
  "1-1": [
    {
      "과목명": "국어",
      "교과군": "국어",
      "과목유형": "공통",
      "학점": 3,
      "원점수": 85,
      "과목평균": 72.3,
      "성취도": "A",
      "성취도별분포비율": "15.2/25.3/30.1/20.0/9.4",
      "석차등급": 1,
      "수강자수": 150
    }
  ],
  "1-2": [...],
  "2-1": [...],
  "2-2": [...],
  "3-1": [...],
  "3-2": [...]
}
```

## 주의사항
- 학년-학기 키는 "1-1", "1-2", "2-1", "2-2", "3-1", "3-2" 형식
- 진로선택과목은 석차등급이 없으므로 null로 표기
- 성취도별분포비율은 "A비율/B비율/C비율/D비율/E비율" 슬래시 구분 문자열
- 숫자 값은 number 타입, 텍스트는 string 타입으로 출력
- 해당 학기에 데이터가 없으면 빈 배열 []
- 2022 개정교육과정 5등급제 기준

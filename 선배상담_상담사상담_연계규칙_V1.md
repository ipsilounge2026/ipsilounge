# 선배 상담 ↔ 상담사 상담 연계 규칙

---

## 1. 연계의 필요성

### 1-1. 두 상담의 관계

본 프로그램은 한 학생이 한 학기에 **두 번의 상담**을 받는 구조:

```
[학기 초 선배 상담 (30분)]   →    [학기 중간: 학생의 실행]    →    [학기 말 상담사 상담 (50분)]
       ↑                                                                    ↓
       └──────────────── [다음 학기로 이어짐] ←───────────────────────┘
```

| 시기 | 상담자 | 소요 | 역할 |
|------|-------|------|------|
| 학기 초 | 졸업 선배 | 30분 | 현장 정보 공유, 방향 제시 |
| 학기 말 | 전문 상담사 | 50분 | 데이터 분석, 전략 확정 |

### 1-2. 왜 연계가 필요한가

**학생 입장:** 매번 같은 이야기를 반복하지 않아도 됨. 한 학원 안에서 일관된 관리를 받는다는 경험.

**선배 입장:** 지난 상담사 상담 이후 학생의 학업 상황이 어떻게 변했는지 알고 시작하면 더 정확한 조언 가능.

**상담사 입장:** 학교 내부 분위기, 선생님과의 관계, 친구 관계 등 데이터로 파악되지 않는 맥락을 선배 기록에서 얻을 수 있음.

**공통 위험:** 연계가 없으면 같은 학생에게 **모순된 조언**이 나갈 수 있음. 예를 들어 선배는 "내신 올인"을 권했는데 상담사는 "수능 중심"을 권하는 경우.

## 2. 연계 기본 원칙

### 2-1. 비대칭 공유 원칙

양방향이지만 **공유 범위가 다름**.

| 방향 | 원칙 |
|------|------|
| 선배 → 상담사 | **전체 공유** (관리자 검토 후) |
| 상담사 → 선배 | **선택적 공유** (항목별 공유 범위 사전 설정) |

**왜 비대칭인가?**

- 선배는 **대학 졸업생 또래**이므로 민감 정보 접근 제한 필요
- 상담사는 **전문가**이므로 더 많은 맥락을 받아도 안전
- 선배가 학생의 심리 상태나 가정 문제를 아는 것은 부적절
- 반대로 상담사가 학생의 학교 생활 맥락을 아는 것은 상담 품질 향상에 직결

### 2-2. 추상화 원칙

상담사 → 선배 공유 시, 구체 데이터는 **추상화된 레이블**로 변환.

| 원본 (상담사 보유) | 선배에게 공유되는 형태 |
|------------------|----------------------|
| 내신 평균 2.3등급 | "상위권" |
| 모의 국어 3등급, 수학 2등급, 영어 1등급 | "균형형" 또는 "수능 대비 강점" |
| D8 심리·컨디션 점수 4/10 | **공유 안 함** |
| E2 목표 대학 "연세대 경영" | "상위권 목표" |
| F 학부모 갈등 서술 | **공유 안 함** |

### 2-3. 연속성 원칙

각 상담은 **이전 상대 상담의 흐름을 이어받음**. 선배는 "지난 상담사 상담 이후 어떻게 됐어?"로 시작할 수 있고, 상담사는 "학기 초에 선배가 동아리 추천해줬는데 어땠어?"로 시작할 수 있음.

이를 위해 각 상담 기록에는 **"다음 상담자에게 전달할 맥락"** 필드가 포함됨.

### 2-4. 관리자 게이트 원칙

모든 공유는 **관리자(학원 운영자)의 검토 후**에 활성화. 자동 공유는 없음.

## 3. 선배 → 상담사 공유 (전체 공유)

### 3-1. 공유 항목

선배 상담 기록 양식의 **모든 항목**이 상담사에게 공유됨 (관리자 검토 후).

| 선배 기록 항목 | 상담사에게 공유됨 | 비고 |
|--------------|----------------|------|
| 기본 정보 | ✅ | 상담사가 이미 알고 있음 |
| 사전 설문 확인 | ✅ | 학생의 관심사 파악에 유용 |
| 이전 상담 체크 포인트 확인 결과 | ✅ | 지속 이슈 추적 |
| **핵심 주제별 진행 결과** | ✅ | 학교 생활 맥락의 핵심 |
| **선택 주제 진행 여부** | ✅ | 학생의 관심사 파악 |
| **자유 질의응답 내용** | ✅ | 학생이 자발적으로 드러낸 고민 |
| **학생 상태 관찰** (분위기, 태도, 특이사항) | ✅ | 데이터로 안 잡히는 맥락 |
| 학생에게 제안한 실천 사항 | ✅ | 이행 여부 추적 |
| 다음 상담 시 확인 필요 사항 | ✅ | 다음 선배 상담에서 체크 포인트로 사용되지만, 상담사도 참고 |
| 학원 운영자 공유 내용 | ⚠ 조건부 | 관리자 판단에 따라 공유 여부 결정 |

### 3-2. 상담사가 얻는 가치

선배 기록에서 상담사가 얻는 **데이터로 잡히지 않는 맥락**:

- 학생의 학교 생활 분위기 (활기·지쳐있음)
- 선생님·동아리·친구 관계의 실제 상태
- 학생이 선배(또래)에게 편하게 드러낸 진짜 고민
- 학원 밖에서의 학생 모습
- 선배가 제안한 실천 사항의 구체적 내용 → 상담사가 이행 여부를 확인 가능

### 3-3. 주의: 상담사도 직접 보지 않는 예외

선배가 작성한 내용 중 다음은 **관리자가 차단** 가능:

- 특정 교사 실명 거론 + 평가성 내용
- 특정 학생 실명 + 관계 갈등 구체 내용
- 학원 내부 이슈 언급
- 선배 본인의 사적 정보

→ 관리자가 검토 시 이 항목들이 포함되어 있으면 **수정 요청 or 해당 부분만 차단**

## 4. 상담사 → 선배 공유 (선택적 공유)

### 4-1. 항목별 공유 규칙

상담사 상담 시스템의 카테고리 A~G 및 상담 결과물을 기준으로 공유 여부 정리.

#### 카테고리 A. 기본 정보

| 항목 | 공유 | 공유 형태 |
|------|------|----------|
| A1 학생 이름 | ✅ | 원본 |
| A2~A6 학교·학년·계열 | ✅ | 원본 (이미 선배가 알고 있음) |

#### 카테고리 B. 내신 성적 ⚠ 추상화

| 항목 | 공유 | 공유 형태 |
|------|------|----------|
| 학기별 과목별 등급 상세 | ❌ | 공유 안 함 |
| **전체 평균 등급대** | ✅ | "상위권(1~2등급)" / "중상위권(3등급)" / "중위권(4등급)" / "중하위권(5등급)" |
| 추세 | ✅ | "상승" / "유지" / "하락" (레이블만) |
| 과목별 상세 | ❌ | 공유 안 함 |

**이유:** 선배가 구체적 점수를 알 필요는 없음. 등급대와 추세 정도면 "이 학생은 어느 정도 수준이고 최근 어떻게 움직이는가"를 파악하기 충분.

#### 카테고리 C. 모의고사 성적 ⚠ 추상화

| 항목 | 공유 | 공유 형태 |
|------|------|----------|
| 회차별 영역별 상세 | ❌ | 공유 안 함 |
| **상담사 유형 판정** (내신형/균형형/수능형) | ✅ | 레이블만 |
| 모의 평균 등급대 | ✅ | "상위권(1~2등급)" 수준 |
| 취약 유형 (C2) | ⚠ 조건부 | 주요 과목 중 1~2개만 요약 ("수학 고난도 취약" 등) |

**이유:** 유형 판정은 선배가 학습 방향 조언을 할 때 유용. "너는 균형형이니까 양쪽 다 관리해야 해" 같은 맥락 제공.

#### 카테고리 D. 학습 현황

| 항목 | 공유 | 이유 |
|------|------|------|
| D1 일주일 학습 스케줄 | ❌ | 너무 개인적 |
| D2 학습 계획 실행력 | ❌ | 선배가 알 필요 없음 |
| D3 오답 관리 습관 | ❌ | 선배가 알 필요 없음 |
| D4 문제 해결 방법 | ❌ | 선배가 알 필요 없음 |
| D5 집중 시간대 | ❌ | 선배가 알 필요 없음 |
| **D6 과목별 학습 고민** | ✅ | "수학 어려워함" 정도는 선배가 경험 공유 가능 |
| **D7 과목별 학습법 요약** | ✅ | 현재 사용 교재·인강 정도만 요약 |
| **D8 심리·컨디션** | ❌ | **민감 정보, 절대 공유 안 함** |

**D7 공유 형태 예시:**
> "수학: 쎈 교재 중심, 메가스터디 인강, 만족도 보통"
> "영어: 현재 학습법에 만족, 교재 유지 중"

선배가 "교재는 괜찮은데 풀이 방식을 바꿔보면 어떨까" 같은 조언을 할 수 있는 수준.

#### 카테고리 E. 진로 & 대입 전략

| 항목 | 공유 | 공유 형태 |
|------|------|----------|
| E1 희망 진로 | ✅ | 원본 ("이공계", "인문사회" 수준) |
| **E2 목표 대학 수준** | ✅ | 수준 레이블만 ("상위권 목표") |
| E2 구체적 학교명 | ❌ | 공유 안 함 (선배의 선입견 방지) |
| **E3 전형 방향** (수시/정시/미정) | ✅ | 레이블만 |
| E4 수능 최저 상세 | ❌ | 너무 전문 영역 |
| E5 선택과목 | ✅ | 리스트 공유 (선배가 과목 조언 가능) |
| E6 학습 우선순위 | ❌ | 선배 역할 밖 |

**왜 구체적 학교명은 공유 안 하는가:**
선배가 "아, 이 학생은 연세대 목표구나" 하고 알게 되면 조언에 편향이 생김. 학생이 부담을 느낄 수도 있음. "상위권 목표"까지만 알고 일반적 조언을 하는 것이 적절.

#### 카테고리 F. 학부모 관점

| 항목 | 공유 |
|------|------|
| F1~F6 모든 항목 | ❌ **절대 공유 안 함** |

**이유:** 가정사는 학생의 프라이버시. 선배가 학부모 정보를 아는 것은 부적절.

#### 카테고리 G. T4 전용 (고3 각오·스케줄·전형)

| 항목 | 공유 |
|------|------|
| G1~G3 모든 항목 | ❌ |

**이유:** T4는 고2-2학기 말인데, 그 다음 학기는 고3 1학기로 선배 상담이 없음. 공유 필요성 자체가 없음.

#### 상담 결과물

| 항목 | 공유 | 공유 형태 |
|------|------|----------|
| **4각형 레이더 (영역별 등급)** | ✅ | 시각화만, 점수 없이 등급 레이블 (고등학교 상담 V3 §4-1: 내신/모의/학습/진로 4영역) |
| 내신·모의 추이 상세 차트 | ❌ | 너무 상세 |
| 내신 vs 모의 비교 | ❌ | 상담사 분석 영역 |
| 과목별 경쟁력 분석 | ❌ | 상담사 분석 영역 |
| 학습 방법 진단 매트릭스 | ❌ | 상담사 분석 영역 |
| **맞춤 전략 로드맵 (최상위 요약만)** | ✅ | Phase별 주요 방향 레이블 |
| 상세 액션 플랜 | ❌ | 상담사 영역 |
| **상담사가 남긴 "다음 선배에게 전달할 맥락"** | ✅ | 별도 필드 신설 (§5 참조) |

### 4-2. 선배가 보게 되는 요약 화면 예시

```

📋 [김학생] 지난 상담사 상담 요약
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
고2-1학기 말 (2025-07-15) · 상담사: 박OO

■ 학업 현황
· 내신 등급대: 상위권 (상승 추세)
· 모의 등급대: 상위권 (유지)
· 학습 유형: 균형형

■ 진로·전형 방향
· 희망 진로: 이공계
· 목표 수준: 상위권 목표
· 전형 방향: 수시 (탐색 단계)

■ 현재 학습 상황
· 고민 과목: 수학 (고난도 문제)

· 미적분 / 물리학 / 화학

· 여름방학 취약 단원 보완 후 2학기 진입

■ 상담사가 선배에게 전달하는 맥락

심리적 요인이 큰 것으로 보임. 수학 이야기를 직접적으로
꺼내기보다 학생이 먼저 말할 때까지 기다려주기 권장."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. "다음 상담자에게 전달할 맥락" 필드 (양쪽 기록에 추가)

### 5-1. 필드의 목적

각 상담자가 **상대 상담자에게 직접 메시지**를 남길 수 있는 필드.
단순 데이터 공유가 아닌, **"이 학생을 어떻게 대해야 하는지"** 경험적 조언.

### 5-2. 선배 → 상담사 맥락 필드

선배 상담 기록 양식에 추가:

```
■ 다음 상담사에게 전달할 맥락 (선택)
ℹ️ 상담사 선생님께 드리는 한마디. 기록 항목에 들어가지 않는
   뉘앙스·분위기·관찰 내용을 자유롭게 남겨주세요.

예시:
· "학교 수업에 흥미를 많이 잃은 것 같음. 동아리는 즐거워함."
· "진로 얘기에 민감하게 반응함. 부모님 기대 부담 크게 느끼는 중."
```

## 7. 데이터 공유 시점 및 프로세스

### 7-1. 시점

```
[상담 종료]
     ↓
[작성자가 상담 기록 작성 완료]
     ↓
[시스템: "검토 대기" 상태로 저장]
     ↓
[관리자 알림]
     ↓
[관리자 검토 및 공유 범위 설정]
     ↓
[검토 완료 → "공유 가능" 상태 전환]
     ↓
[차기 상담 시작 시 상대 상담자에게 자동 노출]
```

**차기 상담 시작 시점**이란:
- 학생이 다음 상담 시점을 선택하고 설문을 시작할 때
- 상대 상담자가 "상담 준비" 화면에 접근할 때

### 7-2. 관리자 SLA

검토 완료까지 **48시간 이내** 권장. 다음 상담 일정까지 여유가 있으므로 긴급하지는 않지만, 검토 누적 방지를 위한 기준.

### 7-3. 검토 미완료 시

관리자 검토가 완료되지 않은 기록은 **다음 상담자에게 노출되지 않음**. 이 경우 해당 상담은 **이전 기록 없이 진행**하게 되므로 품질 저하 우려 → 관리자 업무 중 우선순위 항목.

## 8. 학생·학부모 동의

### 8-1. 회원가입 시 통합 동의 (필수)

본 연계 규칙에 따른 데이터 공유 동의는 **이용약관 및 개인정보처리방침에 포함**되어, 회원가입 시점에 한 번에 받는다. **약관 동의 없이는 회원가입 자체가 불가능**하다.

**동의 대상 (회원가입 시 함께 수락):**
- 이용약관 (연계 원칙은 이용약관 §10에 포함)
- 개인정보처리방침 (민감 정보 차단 원칙은 §7의2에 포함)

**약관에 명시된 핵심 내용:**

```
[상담 데이터 연계 안내 — 이용약관 §10 / 개인정보처리방침 §7의2]

본 서비스는 학기 초 선배 상담과 학기 말 상담사 상담이
연계되어 운영되며, 아래 원칙에 따라 상담 기록이 공유됩니다.

✓ 선배 상담 기록 → 상담사: 관리자 검토 후 참고 자료로 제공
✓ 상담사 상담 기록 → 선배: 관리자 검토 후 추상화된 레이블 형태로만 제공

✓ 다음 항목은 시스템적으로 선배에게 절대 공유되지 않습니다 (관리자도 해제 불가):
  - 심리·정서 상태 (D8)
  - 학부모·가정 관련 내용 (F)
  - 고3 진입 직전 전문 상담 기록 (G)
  - 상세 성적 데이터
  - 구체적 목표 학교명 (기본 비공개 토글)
  - 상세 액션 플랜 (기본 비공개 토글)

✓ 회원은 마이페이지 "선배상담 연계 관리"에서
  개별 상담 기록의 선배 공유를 언제든 중단·복구할 수 있습니다 (부분 철회).
  전면 철회는 회원 탈퇴를 통해서만 가능합니다.
```

### 8-2. 동의를 거부하는 경우

- **회원가입 자체가 불가능하며 서비스 이용이 차단됨**
- 약관 전체 동의 체크 없이는 회원가입 폼 제출 불가 (프론트+백엔드 이중 검증)
- 본 서비스는 상담 연계가 핵심 설계이므로 부분 동의를 허용하지 않음

### 8-3. 학부모 동의

미성년자이므로 학부모 동의도 필요. 학부모 회원은 별도 계정으로 가입하며, 본인 회원가입 시 동일한 약관에 동의한다. 자녀와의 연결은 가족 연결(family_link) 시스템을 통해 사후에 이뤄진다.

### 8-4. 가입 후 부분 철회

회원 지위를 유지한 채 **개별 상담 기록의 선배 공유만 중단**하고 싶은 경우, 마이페이지 "선배상담 연계 관리" 페이지에서 건별·일괄 중단·복구가 가능하다. 이는 연계 동의 자체의 철회가 아니라 이미 공유된 개별 데이터의 사후 비노출 처리에 해당한다. (연계규칙 V1 §10-1 참조)

## 10. 예외 사항 및 주의사항

### 10-1. 학생 동의 철회 시

- 동의 철회 시점 **이후 작성되는** 기록은 공유되지 않음
- 이전에 이미 공유된 기록은 회수 불가 (이미 다음 상담자가 봤을 수 있음)
- 철회 사실은 관리자에게 알림

### 10-2. 민감 정보 사후 발견

관리자 검토 단계에서 놓친 민감 정보가 사후 발견된 경우:
- 해당 기록의 공유 범위 즉시 재조정
- 이미 열람한 상담자에게 "해당 부분은 언급 금지" 안내
- 기록 자체는 원본 보존, 공유 버전만 수정

### 10-3. 선배와 상담사의 의견 불일치

선배 상담과 상담사 상담이 서로 다른 방향을 제시한 경우:
- 기본적으로 **상담사 의견이 우선** (전문성 기반)
- 관리자가 두 기록을 비교하여 학생에게 혼란이 없도록 조정
- 필요 시 학생에게 별도 안내

### 10-4. 데이터 보관 기간

- 상담 기록 원본: 학생이 프로그램에 있는 동안 보관

## 운영 유의사항

1. **암호화 menuurl 토큰** (중앙대·경희대·건국대·인천대): 사이트 메뉴 개편 시 URL 변경 가능 → 연 1회 점검 대상
2. **서울시립대**: URL에 연도가 포함(`/guidebook/2025.do`) → 매년 갱신 필요한 유일한 케이스
3. **명칭 다양**: "학생부종합 가이드북" 외에 "학생부위주전형 가이드북"(교과+종합 통합, 여대·국립대 다수), "워크북"(전남대), "안내서"(이화여대) 등
4. 게시 시기: 대체로 4~7월에 신학년도판 게시 (2026-06 시점 2027판 확인: 연세·경희·외대·동국·홍익·이화·숭실·국민·인하·성신·덕성·인천·ERICA·강원·상명)

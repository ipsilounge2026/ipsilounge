# 고등학교 재학생 과목별 학습 상담 시스템 기획서 (v3)

---

## 1. 시스템 개요

### 1-1. 목적
- 고등학교 재학생의 **현재 학업 상태를 정밀 진단**하고 과목별 학습 전략을 수립
- 내신·모의고사 데이터와 **학습 방법 진단**을 결합하여 성적 부진의 원인을 분석
- 상담 시점(고1-1학기 말 / 고1-2학기 말 / 고2-1학기 말 / 고2-2학기 말)에 따라 **적응형 설문**을 제공

### 1-2. 대상 및 상담 시점

| 시점 | 시기 | 핵심 상담 테마 |
|------|------|---------------|
| **T1** | 고1-1학기 종료 후 (7월) | 고등학교 적응 점검, 첫 내신·모의고사 분석, 학습 방법 전환 진단 |
| **T2** | 고1-2학기 종료 후 (2월) | 1년 성적 추이 분석, 선택과목 학습 준비도, 전형 방향 초기 탐색 |
| **T3** | 고2-1학기 종료 후 (7월) | 선택과목 첫 성적 분석, 수시/정시 방향 탐색, 수능 대비 체계 점검 |
| **T4** | 고2-2학기 종료 후 (2월) | 2년치 종합 진단, 수시/정시 최종 확정, 고3 학습 로드맵 확정 |

### 1-3. 선배 상담과의 역할 분담

본 시스템은 **"학기 초 선배 상담 + 학기 말 상담사 상담"**의 이원 구조 중 **학기 말 상담사 상담**을 담당.

| 선배 상담 (학기 초) | 상담사 상담 (학기 말) |
|-------------------|---------------------|
| 방향 제시·경험 공유 | **결과 분석·전략 확정** |
| 앞으로 뭘 준비할지 | **이번 학기 어땠는지** |
| 일반적 조언 | **개인 데이터 기반 맞춤** |
| 내부자 정보 | **전문적 진단** |

### 1-4. 상담 시간

**회당 50분**

```
00:00 - 03:00  사전 확인 (이전 상담·설문 체크, 인사)
03:00 - 38:00  핵심 주제 진행 (주제당 6~7분)
38:00 - 47:00  자유 질의 + 선택 주제
47:00 - 50:00  마무리 (액션 플랜 요약, 다음 상담 안내)
```

### 1-5. 상담 방식

| 방식 | 진행 | 리포트 공유 |
|------|------|-------------|
| **대면** (기본) | 학원에서 직접 만나 상담 | 상담사가 노트북/태블릿으로 리포트 화면을 보여주며 진행 |
| **비대면** (선택) | Zoom/Google Meet 화상 상담 | 상담사가 화면 공유로 리포트를 보여주며 진행 |

- 기본값은 **대면**, 비대면은 학생/학부모가 방문이 어려운 경우 선택
- 비대면 시 관리자/상담사가 상담 예약에 화상 링크를 입력하고, 학생에게 앱/웹으로 링크 전달
- 화상 도구는 별도 연동 없이 **링크 공유 방식**으로 운영 (Zoom/Google Meet 등 자유 선택)

### 1-6. 활용 흐름
```
[상담 시점 선택(T1/T2/T3/T4)] → [적응형 설문] → [자동 분석·리포트] →
[상담사 리뷰] → [50분 상담 (대면/비대면)] → [맞춤 학습 전략 전달]
```

### 설문 타입: Full / Delta

학생의 상담 이력에 따라 **2가지 설문 타입**으로 운영.

| 타입 | 대상 | 소요 시간 |
|------|------|----------|
| **Full** | 처음 상담받는 학생 (신규·중도 진입 모두) | 30~45분 |
| **Delta** | 이전 상담 이력이 있는 학생 | 10~15분 |

#### 자동 판정 로직

```
[학생이 상담 시점 선택]
         ↓
[시스템: 이전 상담 이력 조회]
         ↓
  ┌─────────────┴──────────────┐
  ↓                             ↓
[이력 없음]                  [이력 있음]
  ↓                             ↓
[Full 설문]                   [Delta 설문]
```

#### Full 설문의 특징

- 전체 카테고리(A~F, T4의 경우 G 포함) 모두 입력
- **성적 데이터(B, C)는 고1-1학기부터 현재까지 모든 학기 입력**
  - 예: T3에 처음 상담하는 학생 → 고1-1학기 + 고1-2학기 + 고2-1학기 성적 모두 입력
- **과거 시점의 학습법·심리·습관(D, E)은 회고 입력하지 않음**
  - 과거 기억은 왜곡되기 쉽고 현재 시점 상담에 큰 의미가 없음
  - 현재 시점의 상태만 입력

#### 성적 입력 칸 자동 생성

사용자가 상담 시점(A4)을 선택하면, 입력해야 할 학기가 자동으로 생성됨:

| 상담 시점 | 자동 생성되는 내신 입력 칸 |
|----------|------------------------|
| T1 | 고1-1학기 |
| T2 | 고1-1학기, 고1-2학기 |
| T3 | 고1-1학기, 고1-2학기, 고2-1학기 |
| T4 | 고1-1학기, 고1-2학기, 고2-1학기, 고2-2학기 |

- **Full 설문:** 해당 학기 칸이 모두 비어있는 상태로 제공 (전부 입력 필요)
- **Delta 설문:** 이전 상담에서 입력한 학기는 자동으로 채워져 있고, 새 학기만 비어있는 상태로 제공

모의고사(C1) 입력 칸도 동일한 방식으로 자동 생성.

#### Delta 설문 항목별 처리 방식

| 카테고리 | Delta 처리 방식 | 근거 |
|---------|---------------|------|
| **A1~A6** | "변경 없음" 체크 → 변경 시에만 재입력 | 거의 변하지 않음 |
| **B (내신)** | **새 학기만 입력 (필수)** | 누적 데이터 |
| **C1 (모의)** | **새 회차만 입력 (필수)** | 누적 데이터 |
| **C2 (취약 유형)** | 매번 재입력 (필수) | 시점마다 달라질 수 있음 |
| **C2-1 (시험 전략)** | T3·T4에서 매번 재입력 | 시점별 중요 |
| **D1 (주간 스케줄)** | "크게 바뀐 점 있음/없음" → 있으면 재입력 | 느리게 변함 |
| **D2 (학습 계획)** | "이전과 동일/변경" → 변경 시 재입력 | 느리게 변함 |
| **D3 (오답 관리)** | "이전과 동일/변경" → 변경 시 재입력 | 느리게 변함 |
| **D4 (문제 해결)** | "이전과 동일/변경" → 변경 시 재입력 | 느리게 변함 |
| **D5 (집중 시간)** | "이전과 동일/변경" → 변경 시 재입력 | 거의 변하지 않음 |
| **D6 (과목 고민)** | **매번 재입력 (필수)** | 시점마다 달라짐 |
| **D7 (학습법)** | **"학습법 변경 여부" 재확인 (필수)** | 본 시스템 핵심 |
| **D8 (심리·컨디션)** | **매번 재입력 (필수)** | 현재 상태 |
| **E1 (진로)** | "이전과 동일/변경" → 변경 시 재입력 | 점진적 변화 |
| **E2 (목표 대학)** | "이전과 동일/변경" → 변경 시 재입력 | 점진적 변화 |
| **E3 (전형 방향)** | **재입력 (필수)** | T2→T3→T4 단계별 진화 |
| **E4 (수능 최저)** | **T2부터 매번 재입력 (필수)** | 시점별 중요 |
| **E5 (선택과목)** | **재입력 (필수)** | 시점별 상태 다름 |
| **E6 (학습 우선순위)** | **재입력 (필수)** | 변화 반영 |
| **G (T4 전용)** | **T4에서는 항상 Full 입력** | T4에서 처음 등장 |
| **F (학부모)** | "이전과 동일/변경" → 변경 시 재입력 | 선택적 |

#### D7 (학습법) 변경 여부 확인 화면 예시 (Delta)

```
[이전 상담에서 입력하신 내용]

📘 수학 학습 방법
· 주요 공부 방식: 문제집 반복 풀이, 인강 수강
· 사용 교재: 쎈, 블랙라벨
· 인강: 메가스터디 ○○○ 강사
· 만족도: 보통

이전 상담 이후 학습 방법을 바꿨나요?
○ 그대로 유지
○ 일부 변경 → [변경된 부분만 수정하기]
○ 완전히 바꿈 → [새로 입력하기]
```

### 카테고리 B. 내신 성적 ●●●●

#### B1~B4: 학기별 교과 내신

입력 학기는 상담 시점(A4)에 따라 자동 생성됨. Full/Delta에 따른 입력 방식(빈 칸 vs 채워진 칸)은 앞의 **"설문 타입: Full / Delta"** 섹션 참조.

**입력 방식 (학기당, 과목별):**

| 필드 | 설명 | 필수 |
|------|------|------|
| 교과 | 국어/수학/영어/사회/과학/기타 중 선택 | ✅ |
| 과목명 | 선택한 교과에 해당하는 과목 목록에서 선택 | ✅ |
| 석차등급 | 1~5등급 (5등급제) | ✅ |
| 성취도 | A~E | ✅ |
| 원점수 | 0~100 | ✅ |
| 과목평균 | 0~100 | ✅ |
| 수강자 수 | 해당 과목 수강 인원 | ✅ |
| 지필:수행 비율 | 100:0 / 80:20 / 70:30 / 60:40 / 50:50 / 기타 | ✅ |

**교과 → 과목 선택 구조:**

| 교과 | 과목 목록 |
|------|-----------|
| 국어 | 공통국어1 / 공통국어2 / 화법과 언어 / 독서와 작문 / 문학 / 주제 탐구 독서 / 문학과 영상 / 직무 의사소통 / 독서 토론과 글쓰기 / 매체 의사소통 / 언어생활 탐구 |
| 수학 | 공통수학1 / 공통수학2 / 미적분1 / 확률과통계 / 미적분2 / 기하 |
| 영어 | 영어 / 영어I / 영어II / 영어 독해와 작문 / 영어 발표와 토론 / 직무 영어 / 영미 문학 읽기 / 심화 영어 / 영어 미디어 커뮤니케이션 |
| 사회 | 통합사회1 / 통합사회2 / 한국사1 / 한국사2 / 세계시민과 지리 / 한국지리 탐구 / 세계지리 탐구 / 정치 / 법과 사회 / 경제 / 사회문화 / 윤리와 사상 / 생활과 윤리 / 동아시아 역사 기행 / 역사로 탐구하는 현대 세계 등 |
| 과학 | 통합과학1 / 통합과학2 / 물리학 / 화학 / 생명과학 / 지구과학 / 역학과 에너지 / 전자기와 양자 / 물질과 에너지 / 화학반응의 세계 / 세포와 물질대사 / 생물의 유전 / 지구시스템과학 / 행성우주과학 등 |
| 기타 | 제2외국어 / 한문 / 정보 / 기타 직접 입력 |

#### B5: 내신 성적 추이 (자동 산출)

- ① 전과목 평균 등급 추이 (학기별 라인 차트)
- ② 과목 그룹별(국/수/영/탐구) 등급 추이
- ③ 등급 분포 (학기별): 1~5등급 각 과목 수
- ④ 등급 분포 변화 추이 (학기 간 비교)
- ⑤ 추세 자동 판정: 상승 / 유지 / 하락 / 등락
- ⑥ 과목별 지필:수행 비율 현황

### 카테고리 C. 모의고사 성적 ●●●●

#### C1: 모의고사 성적 입력

상담 시점에 따라 입력 가능한 모의고사 회차가 자동 생성됨:

| 시점 | 입력 가능 모의고사 |
|------|-------------------|
| T1 | 3월(고1), 6월(고1) |
| T2 | 3월(고1), 6월(고1), 9월(고1), 11월(고1) |
| T3 | 위 4개 + 3월(고2), 6월(고2) |
| T4 | 위 6개 + 9월(고2), 11월(고2) |

※ Full/Delta에 따른 입력 방식은 내신과 동일 ("설문 타입" 섹션 참조)

**입력 방식 (회차별):**

| 필드 | 국어 | 수학 | 영어 | 탐구1 | 탐구2 |
|------|------|------|------|-------|-------|
| 등급 | 1~9 | 1~9 | 1~9 | 1~9 | 1~9 |
| 원점수 | 0~100 | 0~100 | 0~100 | 0~50 | 0~50 |
| 백분위 | 0~100 | 0~100 | - (절대평가) | 0~100 | 0~100 |

※ 미응시 모의고사는 "미응시" 체크

#### C2: 영역 내 취약 유형 자가진단 ●●●●

| 과목 | 질문: 모의고사에서 주로 틀리거나 어려운 유형은? | 응답 (복수 선택) |
|------|----------------------------------------------|----------------|
| 국어 | | 화법·작문 / 문법(언어) / 현대문학 / 고전문학 / 독서(비문학) / 시간 부족 |
| 수학 | | 기본 계산 실수 / 중반 응용 / 후반 고난도(킬러) / 특정 단원(입력) / 시간 부족 |
| 영어 | | 듣기 / 빈칸추론 / 순서배열·문장삽입 / 어법·어휘 / 장문독해 / 시간 부족 |
| 탐구 | | 개념 이해 부족 / 자료·그래프 해석 / 계산 문제 / 고난도 추론 / 시간 부족 |

#### C2-1: 모의고사 시험 전략 ○○●●

T3·T4에서 활성화 — 수능 대비가 본격화되는 시점에서 시험 운영 전략 점검

| 질문 | 응답 |
|------|------|
| 모의고사 시간 내에 모든 문제를 풀 수 있나요? | 매번 시간 부족 / 가끔 부족 / 대체로 여유 |

#### C3: 모의고사 추이 (자동 산출)

### 4-8. 상담사 검토/수정 범위

**기본 원칙:**
- 학생 설문 데이터 기반 자동 분석·초안 생성 → 상담사가 **모든 초안 내용을 검토·편집** 가능 → 최종 리포트 확정
- 상담사가 수정한 내용이 최종 리포트에 반영됨

**상담사 편집 가능 항목:**

| 구분 | 편집 대상 | 편집 방식 |
|------|-----------|-----------|
| 점수/등급 | 4개 영역별 자동 산출 점수·등급 | 점수 직접 수정 → 등급 자동 재산출 |
| 레이더 차트 | 점수 수정 시 자동 반영 | 점수 수정으로 간접 편집 |
| 내신 등급 추이 분석 코멘트 | 자동 생성된 추이 해석 | 텍스트 직접 편집 |
| 모의고사 추이 분석 코멘트 | 자동 생성된 추이 해석 | 텍스트 직접 편집 |
| 내신 vs 모의고사 비교 분석 | 자동 생성된 비교 코멘트 | 텍스트 직접 편집 |
| 과목별 경쟁력 분석 코멘트 | 자동 생성된 과목별 진단 | 텍스트 직접 편집 |
| 학습 방법 진단 코멘트 | 자동 생성된 습관·전략 진단 | 텍스트 직접 편집 |
| 맞춤 전략 로드맵 (4-7) | Phase×4트랙 자동 초안 | 각 셀 내용 자유 편집 (추가/삭제/수정) |

**상담사 코멘트 (리포트 미반영):**

| 항목 | 용도 | 표시 위치 |
|------|------|-----------|
| 상담사 메모 | 상담 시 확인할 포인트, 주의사항, 질문 사항 등 | 상담사 페이지에서만 표시 |
| 상담 전 체크리스트 | 이 학생 상담 전 미리 검토할 사항 정리 | 상담사 페이지에서만 표시 |

- 상담사 코멘트는 상담 준비·진행용이며, 학생/학부모에게 전달되는 리포트에는 **포함되지 않음**
- 상담사 페이지에서 리포트 내용과 함께 상담사 코멘트가 나란히 표시되어, 상담 시 참고 가능

**상담 기록 불변 정책:**

> 위의 "상담사 편집 가능 항목"은 **사전 설문 기반 자동 분석 리포트**에 한정됨.
> **상담 기록** (상담사가 상담 내용을 기록한 데이터)은 아래 규칙을 따름:

| 구분 | 정책 |
|------|------|
| 최초 작성 | 상담 후 1회 작성 |
| 수정 | **불가** — 작성 완료된 기록은 어떤 필드도 수정할 수 없음 |
| 삭제 | **불가** — 작성된 기록은 삭제할 수 없음 |
| 추가 기록 | **가능** — 기존 기록에 보충 내용을 추가 작성할 수 있으며, 추가 기록에는 작성자·작성일시가 자동 기록됨 |
| 이력 보존 | 최초 기록 + 모든 추가 기록이 시간순으로 보존되어 전체 이력을 확인할 수 있음 |

- 상담 기록의 불변성은 상담 품질 관리 및 법적 증빙 목적으로 필수 요건임
- 공개/비공개 설정만 변경 가능 (학생에게 보여줄지 여부)

### 9-3. 선배 상담과의 연계

본 시스템은 **학기 초 선배 상담**과 연계되어 운영됨.

상세 규칙은 별도 문서 참조: **『선배상담_상담사상담_연계규칙.md』**

**핵심 요약:**

**데이터 흐름:**
```
학기 초 선배 상담 (30분) → 학기 말 상담사 상담 (50분) → 다음 학기 초 선배 상담 → ...
```

**공유 원칙:**
- **선배 → 상담사:** 선배 상담 기록 **전체 공유** (관리자 검토 후)
  - 학교 생활 맥락, 학생 분위기, 자유 질의 내용 등 데이터로 잡히지 않는 맥락 제공
- **상담사 → 선배:** **추상화된 형태로 선택적 공유**
  - 공유 O: 학업 등급대 레이블, 학습 유형(내신형/균형형/수능형), 진로 방향, 선택과목, D6 고민, D7 학습법 요약
  - 공유 X: **D8 심리·컨디션, F 학부모, 상세 성적, 구체적 목표 학교명, 상세 액션 플랜, G 고3 준비**

**상담사 상담 기록에 추가되는 필드:**

```
■ 다음 선배에게 전달할 맥락 (선택)
ℹ️ 다음 학기 초 선배 상담을 맡을 선배에게 드리는 한마디.
   학생과 대화 시 주의할 점이나 유의 사항.

예시:
· "수학 이야기 직접 꺼내지 말 것. 학생이 먼저 꺼낼 때까지 대기."
· "목표 대학 언급 자제. 부담 큰 상태."
```

**관리자 역할:**
- 모든 공유는 관리자 검토 후 활성화
- 일부 민감 항목(D8, F, G, 상세 성적)은 **시스템적으로 공유 차단** (관리자도 해제 불가)

**학생·학부모 동의:**
- 프로그램 등록 시 데이터 연계 동의 필요
- 거부 시 두 상담은 독립 운영
